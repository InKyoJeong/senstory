## Senstroy

- Sense + Story
- 사람들과 소통하는 공간과 나만의 감정을 기록하는 공간을 더한 sns 프로젝트

<br>

## 사용 기술

- Front : react(next), redux-saga
- Back : nodejs(express), sequelize
